## Agent skills

### Issue tracker

Issues live as local markdown files under `.scratch/<feature-slug>/`. External PRs are not a triage surface (no git remote). See `docs/agents/issue-tracker.md`.

### Triage labels

Uses the default five canonical labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout — one `CONTEXT.md` at repo root + `docs/adr/` for architectural decisions. See `docs/agents/domain.md`.
